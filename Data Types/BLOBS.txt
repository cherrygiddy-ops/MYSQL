BLOBS ->USED TO STORE LARGE AMOUNT OF BINARY DATA LIKE PDFS ,IMAGES,VIDEOS,WORK FILES

TINYBLOB -> 255BYTES
BLOB ->65KB
MEDIUMBLOB ->16 MB
LONGBLOB - 4 GB

N/B  -> RELATIONAL DATABSE USED FOR RELATIONAL DATA NOT BINARY DATA (STORE BINARY DATA IN FILE SYSTEM)

PROBLEMS WITH STORING FILES IN A DATABASE
1.INCREASED DATABASE SIZE
2.SLOWER BACKUP
3.PERFORMANCE PROBLEM ->FASTER WHEN READING IMAGE ON THE FILE SYSTEM
4.MORE CODE TO READ /WRITE IMAGE IN DB
